简介

MLSQL Console 使用入门

MLSQL Console是MLSQL Stack 套件中非常重要的一款直接面向普通用户的产品。 MLSQL Console支持无编码所见即所得的交互性数据分析，同时也提供了面向大数据和AI的语言MLSQL的支持，拥有诸如脚本管理，代码补全，执行等功能。
