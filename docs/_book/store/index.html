MLSQL 脚本商店

MLSQL 作为脚本语言，提供了很多可复用特性，所以用户可以将自己写的一些脚本提供出来，供其他用户使用。 MLSQL脚本商店目前就是做这个用途的。

如果觉得自己的MLSQL脚本很酷，欢迎到MLSQL Github ISSUE中黏贴出来，我们会放到官方仓库里。

脚本类型的插件其实有很多天然优势：

  1. 无需重启或者部署。修改完即可生效；
  2. 可以多个版本同时存在，只需要使用不同的名字即可。
  3. 结合python代码或者是scala/java UDF/UDAF，也可以实现很多复杂的功能。
