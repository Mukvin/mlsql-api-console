MLSQL Stack 2.0.0 Bug 合集

MLSQL Stack 2.0.0 Bug 合集

该文档主要记录2.0.0版本已知bug,包括Console/Engine. 我们会在2.0.1版本修复。用户如果需要也可以直接使用 2.0.1-SNAPSHOT。

MLSQL Console

  1. [Console][2.0.0][Bug] PacketTooBigException
  2. [Console][2.0.0][Bug][AanalysisWorkshop] 修改字段名时，不能修改后和原来的名称相同
  3. [Console][2.0.0][Bug] 添加Engine无法删除， 在Console里无法正确选择Engine
  4. [Console][2.0.0][Bug] 控制台脚本目录 expand/collapse 的问题
  5. [Console][2.0.0][Bug] 上传文件时，使用配置而非数据库主目录配置
  6. [Console][2.0.0][Bug] 下载个人目录文件提示没有权限

MLSQL Engine

  1. [Engine][2.0.0][Bug][Ray] 结合Ray运行长任务时，会出现Connection Refused问题
  2. [Engine][2.0.0][Bug] 使用!ray,!python命令行权限控制问题
  3. [Engine][2.0.1][Feature] 支持读取二进制文件
  4. [Engine][2.0.1][Feature] 支持保存二进制数组到HDFS文件系统
  5. [Engine][2.0.1][Bug] 解决在spark-3.0中运行!ray 会出现jackson-scala版本问题
  6. [Engine][2.0.1][Bug] !ray指令中如果配置运行地址为driver时会出现socket相关问题
  7. [Engine][2.0.1][Bug] WowRowEncoder 导致Python获取Spark数据非常慢
  8. [Engine][2.0.0][Feature] 增加engine sparkcontext监控接口
