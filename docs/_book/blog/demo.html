Excel 支持 (2020-09-01)

增加excel插件支持，支持在MLSQL中使用python处理excel. 上传你的excel尝试下吧。 结合Python读取Excel

或者使用如下方式加载excel:

load excel.`/tmp/upload/测试.xlsx` 
where useHeader="true" 
and maxRowsInMemory="100"
as output;

保存：

save overwrite table1 as excel.`/tmp/upload/测试-1.xlsx` 
where useHeader="true";

如何使用

MLSQL Console 现在是自助的 注册和登录的。首先第一次先点击注册啦：

注册完成后直接就进去了。

进去后，我们提供了示例数据：

如果在【分析工坊】利用，直接点击右键，选择打开即可。 如果需要在【控制台】使用，黏贴复制运行如下命令即可：

load delta.`public.example_data` as cc;

Console的使用文档看这里：MLSQL Console 官方文档
